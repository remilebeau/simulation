# Simulation

A web app that features the following business analytics tools:

- Monte Carlo simulation for production planning
- Monte Carlo simulation for financial planning
- Monte Carlo simulation for cash flow

## Created with:

- Next.js
- Tailwind CSS
- ShadCN
- Plotly.js
- FastAPI
